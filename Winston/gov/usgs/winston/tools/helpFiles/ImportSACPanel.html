Winston ImportSAC
This program imports data from SAC files into a Winston database. Information about connecting to the Winston database must be present in Winston.config in the current directory. 
% java -cp lib/winston.jar gov.usgs.winston.in.ImportSAC
RSAM parameters: delta=10, duration=60.
