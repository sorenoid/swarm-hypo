Import Instrument Locations
This program imports instrument locatoins from a dataless SEED volume. Information about connecting to the Winston database must be present in Winston.config in the current directory. 
% java -cp lib/winston.jar gov.usgs.winston.in.ImportDataless
Winston ImportDataless

Usage:
  java gov.usgs.winston.in.ImportDataless [-c <winston.config>] <dataless>
