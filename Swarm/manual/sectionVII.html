Section VII, The Menu

Note: the menu in Swarm is likely to undergo significant changes in the next few versions.

Here is a breakdown of what you'll find on the Swarm menu bar:

File

    Exit - exit Swarm

Edit
    Options... - edits application-wide settings, current only time zone settings.

Window
    Tile Helicorders - tiles all open helicorders.
    Tile Waves - tiles all open waves.
    Kiosk Mode - enters Kiosk mode.

Help
    About... - opens the About box which displays some information about memory usage.
