Introduction
Swarm is a program for the real-time analysis of seismic waveform data. Currently it can load data from Earthworm Wave Servers, Winston Wave Servers, SEED volumes, and SAC files.

This is a beta version of Swarm. There are missing features, known bugs, and other limitations to this version.
Please email me bug reports or feature requests. No bug report is too minor (such as a minor UI quirk). My email address is: dcervelli@stanford.edu

The current version is: 1.2.1.20051001
Important note #1: Swarm now REQUIRES Java 1.5.

Important note #2: While some sections of the manual have been updated, there are many that are quite out of date.

Important note #3: the fixes from 1.2.0 -> 1.2.1 -> 1.2.2 are minor for most users.

A list of bug fixes and changes can be found in Appendix C of the manual.

Installation
1. Install Java Runtime Environment (or SDK) 1.5 (also known as Java 5.0). Available at http://java.sun.com/j2se/1.5.0/index.jsp

2. Download Swarm.
zip: swarm-bin.zip
tar: swarm-bin.tar.gz

3. Unzip or untar. The program files are located under a directory called 'swarm' so it is safe to unzip wherever you like.

4. Run Swarm. On Windows, simply run 'swarm.bat' in the 'swarm' directory. On Unix-like platforms, run the 'swarm.sh' script.

5. Go through the tutorial. This will quickly get you started using Swarm.

The manual is available online.
