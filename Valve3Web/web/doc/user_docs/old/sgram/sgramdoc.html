Spectrogram Section User's Guide

The main Spectrogram UI looks something like this (depending on your Valve installation):

The interface is extremely simple. Simply select a station (control-click/shift-click for multiple stations, one plot is produced for each station) and a time interval (due to the relative complexity of generating these plots, a maximum time span of one month is enforced) and press submit. Valve is not generating Spectrograms on-the-fly, the data are being imported from another process: the earthworm sgram module.
