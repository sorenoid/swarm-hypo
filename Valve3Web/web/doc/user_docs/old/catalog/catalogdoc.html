Catalog Section User's Guide

The main Catalog UI looks something like this (depending somewhat on your Valve installation):

The Catalog section get generate many different types of plots: map plots (including cross-sections), earthquake counts time-series, 3-D hypocenter plots, and earthquake density plots. In order to provide this flexibility the UI is unfortunately a bit complex.

First, for all plots an area has to be selected. The Area of Interest provides the mechanism for selecting the area. The values entered in the (W/N) and (E/S) fields, depth fields, and magnitude range are used to select the proper earthquakes for visualization. Entering longitudes and latitudes by hand is cumbersome so the pull-down box ('[Select a region...]' above) provides a list of predetermined areas that fill in the longitude/latitude fields for you. In fact, you should never have to enter the values by hand unless you specifically want to.

After the area has been selected the output type has to be chosen.

Hypocenter Map Plots
There are two main types of hypocenter maps: actual hypocenter plots and hypocenter density plots. Each hypocenter is plotted individually on the standard hypocenter plot while the counts of hypocenters in a spatial cell are plotted in the density plots.

Standard Hypocenter Map Plots
All of the hypocenter map plots have two things in common: at least one of their axes is spatial and the body of the plot is hypocenters. The many options available for the hypocenter map plots make them quite versatile. The most important option is the axes designation. The options are:

  • Lon-Lat: longitude and latitude on the x and y axes respectively, the standard map plot.
  • Lon-Depth: longitude on the x axis and depth on the y axis, the preferred standard cross-section plot.
  • Lat-Depth: latitude on the x axis and depth on the y axis, another cross-section plot.
  • Arbitrary-Depth: an arbitrary, user-defined cross-section on the x axis, depth on the y axis.
  • Arbitrary-Time: an arbitrary, user-defined cross-section on the y axis, time on the x axis. This is actually a time-series and with respect to the Valve UI behaves as such.

When using arbitrary cross-sections, the cross-section is defined by the points specified by the A and A' text fields. The easiest way to select these points is to generate a standard map plot wholly encompassing your area of interest, select "Map clicks select: Cross-section", then click the two points that define the cross-section letting Valve enter the coordinates for you. If you want to limit the width of the cross-section, specify the width in the appropriate text field (the width is on either side, so if you wanted a 5km total width, enter 2.5). Note: the Area of Interest filter is still in effect when plotting hypocenters with arbitrary cross-sections, so if the cross-section is near the border of the original map you may not be getting hypocenters that fall within the width of the cross-section.

Of course, the "Hypocenters" radio box needs to be checked to generate a hypocenter plot. The last option available for hypocenter plots is whether color should represent depth or time.

Hypocenter Density Plots
Hypocenter density plots work similarly to hypocenter plots except that the number of hypocenters that fall within a grid cell (sized depending on the "Bin" setting) is plotted as a color map. Arbitrary cross-sections are not allowed with hypocenter density plots. Also, you can specify whether the color map should be based on raw counts or the common log of the raw counts.

Hypocenter Count Plots
Valve can create hypocenter count time-series plots. The Area of Interest filter is still used for these plots, so you can quickly get historical earthquake counts at a standard region over any time interval. Additionally, a cumulative count, moment, or magnitude line (simply select the desired radio box) is overlaid on the count plot. You can also specify the bin size for the count histogram (a large bin size will be automatically chosen if the user-specified one results in too many bins). Cumulative moment and magnitude are calculated as:
 Mo = 10 ^ ( (321/20) + (3*[Magnitude]/2) )
 Mag = Log(Mo) / 1.5 - 10.7

3-D Hypocenter Plots

Valve uses the LiveGraphics3D applet to interactively display hypocenters in 3-D. You can specify the axes you want in the plot and whether you want the size of the hypocenter to correspond to magnitude and/or the color to correspond to the time.
